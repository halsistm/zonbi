**作成日:** 2026-04-20

| 項目 | 内容 |

| ジャンル | 1人称 3Dスナイパーシューティング |

| レイヤー | 採用技術 | 理由 |

## 3. ゲームの舞台・ワールド設計

### 3-1. マップ概要
- 1人称固定視点（移動なし）。プレイヤーは廃ビルの屋上に固定配置
- 廃墟の街並みが正面に広がる半オープンワールド構造
- ゾンビは複数の路地・廃ビル間から出現し、プレイヤー方向へ接近

### 3-2. 3Dオブジェクト構成
| オブジェクト | 詳細 |
|-------------|------|
| 廃ビル | 低ポリ崩壊ビル 5〜8棟（ランダム配置） |
| 炎上する車 | PointLight＋パーティクル炎エフェクト 3〜5台 |
| 瓦礫・障害物 | ゾンビの遮蔽物・移動経路の迷彩として配置 |
| 空 | 夜空 or 夕暮れ（CubeMap or グラデーションSkybox） |
| 炎・煙 | Three.jsパーティクルシステム |

### 3-3. ライティング
- AmbientLight（薄暗い環境光）
- 炎上車からの PointLight（オレンジ系動的揺らぎ）
- 月光 DirectionalLight（影付き）

## 4. カメラ・視点仕様

| 項目 | 仕様 |
|------|------|
| 視点 | 1人称（FPS）固定位置 |
| 通常視野角 | FOV 75° |
| スコープ視野角 | FOV 15°（約5倍相当） |
| カメラ操作 | PC：マウス移動で水平スイング（左右のみ、垂直は限定的） |
| カメラ操作 | スマホ：画面スワイプで同様 |
| 水平回転範囲 | ±160°（背後は向けない） |
| スコープ中の挙動 | 画面中央に照準固定・周囲がマスク（黒枠）で覆われる |

## 5. ゾンビ仕様

### 5-1. 基本動作
- 出現ポイントからプレイヤー方向へ一直線に前進
- 障害物を回り込む簡易経路探索（ウェイポイント方式）
- 出現時に唸り声・足音などのサウンドキュー発生

### 5-2. ダメージ・撃破仕様
| 部位 | 効果 |
|------|------|
| **ヘッド** | 1発即死（ヘッドショット） |
| 胴体 | 複数発でHPを削り撃破（HP：3〜5発分、難易度連動で増加） |
| 四肢（オプション） | 移動速度低下（将来拡張） |

### 5-3. 難易度の時間連動スケーリング
| 経過時間 | 出現数/分 | 移動速度 | ゾンビHP（胴体） |
|---------|----------|---------|-----------------|
| 0〜60秒 | 4〜6体 | 1.0× | 3発 |
| 61〜120秒 | 8〜10体 | 1.3× | 4発 |
| 121〜180秒 | 12〜15体 | 1.7× | 5発 |
| 181秒〜 | 20体+ | 2.5× | 5〜7発 |

- 出現インターバルは時間経過で短縮
- 複数方向から同時出現（後半は全方向から）

## 6. プレイヤー操作仕様

### 6-1. PC操作（マウス＋キーボード）
| 操作 | 入力 |
|------|------|
| 視点回転 | マウス移動（Pointer Lock API） |
| スコープON/OFF | 右クリック or `Z`キー（トグル） |
| 射撃 | 左クリック |
| ゲーム一時停止 | `Esc` |

### 6-2. スマホ操作（タッチ）
| 操作 | 入力 |
|------|------|
| 視点回転 | 画面スワイプ（左右） |
| スコープボタン | 画面右下の固定ボタン（タップでトグル） |
| 射撃 | 画面右下の射撃ボタン（スコープ中のみ有効） |
| ミニマップ確認 | 常時表示（左上） |

### 6-3. スコープ射撃フロー
```
1. ゾンビが視野内で聴覚検知 → 唸り声・足音
2. プレイヤーがスワイプ/マウスでゾンビ方向を向く
3. 画面中央に照準（クロスヘア）が重なる
4. スコープボタンを押す → スコープUI表示
5. 距離に応じた偏差補正を考慮して射撃ボタン
6. 命中判定 → エフェクト再生
```

---

## 7. 射撃・弾道仕様

### 7-1. 弾薬
- 弾数無制限・リロードなし
- 連射制限あり（射撃間隔：0.8秒 / スコープ解除後は0.5秒ペナルティ）

### 7-2. 距離偏差（弾落ち）
| 距離 | 偏差量（照準からのズレ） |
|------|------------------------|
| 0〜30m | なし |
| 31〜60m | 小（照準の5%下） |
| 61〜100m | 中（照準の15%下） |
| 101m〜 | 大（照準の30%下） |

- スコープなしでの射撃は命中率大幅低下（散弾補正）
- スコープ中は距離表示UI（任意実装）

---

## 8. ゾンビ検知・音響システム

### 8-1. 3D音響
- Web Audio API の `PannerNode` を使用
- ゾンビの位置に応じて左右・前後の定位音響を実現
- 足音・唸り声は距離に応じて音量変化（距離減衰）

### 8-2. 音無し対応（ミニマップ）
- 画面左上に常時ミニマップ表示（俯瞰円形）
- 音が鳴ったゾンビの方向を **矢印（→）** でリアルタイム表示
- 矢印は音量に比例した大きさ・距離に応じた透明度
- プレイヤー正面を上として回転表示

### 8-3. 効果音一覧
| 種別 | 内容 |
|------|------|
| ゾンビ足音 | 距離減衰つき3D定位音 |
| ゾンビ唸り声 | 出現時・接近時 |
| 射撃音 | スナイパーライフル発砲音 |
| ヘッドショット | 特殊命中SE |
| 胴体命中 | 通常命中SE |
| 空撃ち（スコープ外） | 警告音 |
| ゲームオーバー | 絶叫・サイレン |

## 9. HUD（ヘッドアップディスプレイ）

```
┌──────────────────────────────────────────┐
│ [ミニマップ]              スコア: 00000  │
│  ↗ ← 矢印              サバイバル: 02:34 │
│                                          │
│              [ + ]  ← クロスヘア         │
│          （スコープ中は照準レンズUI）     │
│                                          │
│ ❤❤❤  見逃し残り: ■■■□□            │
└──────────────────────────────────────────┘
```

| UI要素 | 詳細 |
|--------|------|
| スコア | 右上・リアルタイム更新 |
| 生存タイマー | 右上スコア下 |
| 見逃しゲージ | 左下、許容見逃し数を■で表示（例：5回まで） |
| クロスヘア | 中央固定（スコープ中は照準レンズに変化） |
| ミニマップ | 左上・半透明円形・音方向矢印付き |
| スコープUI | 画面全体マスク＋中央レンズ（スマホ：右下ボタン） |

## 10. スコア・ゲームオーバー仕様

### 10-1. スコア計算
| 行動 | 点数 |
|------|------|
| ヘッドショット撃破 | +200pt |
| 胴体撃破 | +100pt |
| 連続撃破ボーナス（コンボ） | ×1.5〜×3.0 |
| 見逃し（1体） | −50pt（マイナスなし下限0） |

### 10-2. ゲームオーバー条件
- ゾンビを **5体見逃した** 時点でゲームオーバー
- 見逃し = ゾンビがプレイヤー直下（Z距離0）まで到達

### 10-3. スコアランキング（ローカル）
- `localStorage` に上位10件を保存
- ゲームオーバー後にランキング画面を表示
- 名前入力（3文字）→ 登録

## 11. ゲームフロー

```
タイトル画面
  └→ [スタート] → チュートリアル表示（初回のみ）
       └→ ゲームプレイ（エンドレス）
            └→ 見逃し5体でゲームオーバー
                 └→ スコア表示 → ランキング登録
                      └→ [もう一度] or [タイトルへ]
```
